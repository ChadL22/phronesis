## Design Patterns Established

### HTML Structure
```html
<div class="unified-filter-controls">
  <div class="filter-primary-row">
    <div class="filter-primary-section">
      <span class="filter-label">Category</span>
      <button class="filter-btn active">All</button>
      <button class="filter-btn">Option 1</button>
    </div>

    <button class="filter-more-toggle" id="toggle-secondary">
      <svg viewBox="0 0 24 24"><polyline points="6 9 12 15 18 9"></polyline></svg>
      More Filters
    </button>

    <div class="filter-view-toggle">
      <button class="filter-view-btn active">Cards</button>
      <button class="filter-view-btn">List</button>
    </div>
  </div>

  <div class="filter-secondary" id="secondary-filters">
    <div class="filter-secondary-inner">
      <div class="filter-group">
        <span class="filter-label">Secondary</span>
        <button class="filter-btn active">All</button>
      </div>
    </div>
  </div>
</div>
```

### JavaScript Pattern
```javascript
document.getElementById('toggle-secondary').addEventListener('click', function() {
  this.classList.toggle('active');
  document.getElementById('secondary-filters').classList.toggle('open');
});
```

### CSS Behavior
- max-height: 0 (collapsed) → max-height: 600px (open)
- Chevron rotation: transform: rotate(180deg)
- Filter buttons: 0.6rem uppercase with accent color on active
- Smooth 0.28s cubic-bezier(0.4,0,0.2,1) transition

### Mobile Responsive (@900px)
- Primary row: flex-direction: column
- Labels: margin-bottom: 0.3rem (above buttons)
- Buttons wrap horizontally within groups
- More Filters button: width: 100%, centered
- View toggle: width: 100%, flex: 1 on buttons

## Key Learnings

- Always verify live file state before editing (local/deployed can diverge)
- Incremental changes over wholesale rewrites (per user preference)
- DOM-based detection over character-count heuristics for UI behavior
- Prototype first, then implement systematically page-by-page
- Reusable components save time and ensure consistency
